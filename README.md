# Python OOP Practice - Student Grade Management

A Python script demonstrating object-oriented programming concepts with student grade tracking.

## Features

- **Student Class** with name, email, and grades management
- **Grade Operations**: Add grades, calculate averages, display info
- **Data Structures**: Lists, tuples, sets, and dictionaries
- **Email Validation**: Regex-based email format checking
- **Tuple Immutability**: Demonstration with error handling

## Requirements

- Python 3.6+

## Usage

```bash
python python-practice.py
```

## What It Does

- Creates 3 student objects with grades
- Manages grades using list operations (add, pop, access)
- Stores students in a dictionary by email
- Tracks unique grades using sets
- Validates email formats with regex
- Counts grades above 90
